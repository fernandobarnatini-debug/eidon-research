# EIDON Research — Brand Guidelines

## Brand Name
- **Name:** EIDON Research
- **Pronunciation:** EYE-don
- **Origin:** Greek "εἶδον" (I saw) + Biblical "image of God"
- **Slogan:** "See the Form"

## Colors

### Primary
- **Navy (Primary Dark):** #0B1426 — Header, footer, dark accent sections
- **Navy Light:** #111d38 — Hover states, secondary backgrounds
- **Gold (Accent):** #D4AF37 — Buttons, highlights, badges. USE SPARINGLY.
- **Gold Light:** #e8c84a — Hover state for gold elements

### Backgrounds
- **Main Content:** #FFFFFF (white)
- **Alternating Sections:** #f8f9fa (off-white)
- **Card Backgrounds:** #FFFFFF with #e5e5e5 borders

### Text
- **Headings:** #0B1426 (navy) or #1a1a1a (near-black)
- **Body Text:** #333333
- **Secondary/Muted:** #555555
- **Light text (on dark bg):** #FFFFFF

### Accent
- **Electric Blue:** #00D4FF — Used very rarely, for special highlights only

## Typography

### Headings
- **Font:** "DM Sans" (Google Fonts) — Bold (700), ExtraBold (800)
- **Fallback:** -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif
- **Style:** Clean, modern, no decorative fonts

### Body
- **Font:** System font stack: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif
- **Weight:** Regular (400) for body, Medium (500) for emphasis
- **Size:** 16px base, 1.6 line-height

### DO NOT USE as primary font:
- Inter
- Roboto
- Arial

## Logo
- Text-based: "EIDON" in bold DM Sans with "RESEARCH" underneath in lighter weight
- Can be navy on light backgrounds, white on dark backgrounds
- If logo.png is provided in this folder, use that instead

## Voice & Tone
- Professional but not clinical
- Confident, authoritative
- "Research use only" language throughout
- Never casual or slang
- Trust-building: emphasize purity, testing, quality

## Layout
- Max content width: 1200px, centered
- Section padding: 80px vertical minimum
- Card border-radius: 12px
- Button border-radius: 8px
- Generous whitespace everywhere

## Imagery Style
- Clean, minimal
- Product photography style (even if using CSS placeholder graphics)
- Color-coded product categories:
  - Retatrutide → Blue gradient
  - CJC-1295 + Ipamorelin → Green gradient
  - GHK-Cu → Purple gradient
  - Melanotan 2 → Amber/Orange gradient
  - BAC Water → Gray/Silver gradient

## What "Good" Looks Like
- Reference: onyxresearch.shop
- Clean, professional, trustworthy
- Looks like a real research chemical company
- Mobile-first (90% of traffic from Snapchat)
- Someone should think "this is legit" within 2 seconds
