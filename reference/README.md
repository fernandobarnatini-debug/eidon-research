# Reference Screenshot

## How to capture the reference screenshot:

1. Go to https://onyxresearch.shop in Chrome
2. Press F12 (Windows) or Cmd+Option+I (Mac) to open DevTools
3. Press Ctrl+Shift+P (Windows) or Cmd+Shift+P (Mac)
4. Type "screenshot" and select "Capture full size screenshot"
5. Save the downloaded image as `onyx-screenshot.png` in this folder

This screenshot is what Claude Code will use as the structural layout reference.
